OPUS: Object Positioning Utility Script

Opus is a console tool for the testing of position coordinates and rotation vectors. It is quite tedius to have to guess these things during development. Opus understands and is here to help. Hit the backtick (`) to open the Onset console.

Attach object to player

Usage: attach <modelid> <x> <y> <z> <rx> <ry> <rz> <SocketName>

> attach 1271 33 -8 0 360 260 -110 hand_l

Add light component to attached object

Usage: light <x> <y> <z> <rx> <ry> <rz>

> attach 1271 33 -8 0 360 260 -110 hand_l

Detach object and component from player

Usage: detach
