OPUS: Object Positioning Utility Script

/attach modelid bone - Attach an object to player bone

/deattach - Deattach all objects

[F] / SHIFT [F] - Relative X location

[G] / SHIFT [G] - Relative Y location

[H] / SHIFT [H] - Relative Z location

[J] / SHIFT [J] - Relative pitch rotation

[K] / SHIFT [K] - Relative yaw rotation

[L] / SHIFT [L] - Relative roll rotation
