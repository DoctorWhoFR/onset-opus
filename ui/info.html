OPUS: Object Positioning Utility Script

/attach <modelid> <bone> - Attach an object to player bone
/detach - Detach object from player

Once attached, use the keys below to adjust the attachment position. (^ means shift)
The default increment is 10. Use [N]/[^N] to change increments.

[X] / [^X] - Relative X location
[Y] / [^Y] - Relative Y location
[Z] / [^Z] - Relative Z location
[J] / [^J] - Relative pitch rotation
[K] / [^K] - Relative yaw rotation
[L] / [^L] - Relative roll rotation
